## Description
A comprehensive website for aspiring web developers to learn HTML, CSS, and JS.

## Features

- **Become a Web Developer Section:** A welcoming section with an illustration of a female developer and a brief introduction.
- **Newsletter Signup:** Allows visitors to stay updated by subscribing to the newsletter.
- **Learning Options:** Presents different modes of learning - virtual, in front, and in person.
- **Learn The Fundamentals:** Sections dedicated to HTML, CSS, and JS with engaging graphics.
- **Frequently Asked Questions:** A section answering common queries about the bootcamp.
- **Our Instructors:** Meet the experienced team of instructors with their brief profiles.
- **Contact Info:** Provides address, phone number, and email for easy communication.

## Technologies Used
- HTML
- CSS
- JavaScript
  
## Installation

1. Clone this repository to your local machine using:
  ```bash
  git clone https://github.com/VinayShetyeOfficial/Frontend-Bootcamp-Website.git
  ```

2. Navigate to the project directory:
  ```bash
  cd Frontend-Bootcamp-Website
  ```

3. Open `index.html` in your web browser.
